# Stone Site Standard Kit

这是一套石材外贸网站的通用建站规范包，适合复用到新的大理石、石英岩、花岗岩、人造石英石、板材库存、工厂供应商网站。

目标不是做一次性页面，而是做一个能长期维护的纯静态网站：

- 产品、板面、文章数据化。
- 图片先规范，再建页面。
- 页面服务询盘和 SEO，不做空泛展示页。
- 后台可维护产品、板面、文章和图库。
- 默认部署到 GitHub + Cloudflare Pages。

## 使用顺序

1. 复制本文件夹到新项目的 `docs/` 下。
2. 先填写 `00-project-brief-template.md`。
3. 按 `02-image-asset-standard.md` 整理图片。
4. 按 `03-content-model-standard.md` 建产品、板面、文章数据。
5. 按 `04-page-structure-standard.md` 搭页面。
6. 按 `05-seo-article-standard.md` 写 SEO 文章。
7. 按 `06-admin-workflow-standard.md` 配后台。
8. 上线前执行 `07-launch-checklist.md`。

## 文件说明

- `00-project-brief-template.md`：新石材网站开工前要确认的信息。
- `01-build-rules.md`：技术栈、目录、部署、禁止项。
- `02-image-asset-standard.md`：图片分类、命名、尺寸、裁切、图库规则。
- `03-content-model-standard.md`：产品、板面、文章、图库字段规范。
- `04-page-structure-standard.md`：首页、产品页、板面页、工厂页、联系页结构。
- `05-seo-article-standard.md`：SEO 文章模板和排版规则。
- `06-admin-workflow-standard.md`：静态后台和 GitHub Token 操作规范。
- `07-launch-checklist.md`：上线前检查清单。
- `templates/`：可复制的 JSON 内容模板。
