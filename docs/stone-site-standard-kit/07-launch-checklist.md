# 07 Launch Checklist

## 基础

- [ ] 使用纯静态架构。
- [ ] 没有数据库。
- [ ] 没有 SSR。
- [ ] 内容数据化。
- [ ] `robots.txt` 存在。
- [ ] `sitemap.xml` 可访问。
- [ ] 404 页面存在。

## 图片

- [ ] 图片已按分类整理。
- [ ] 文件名是英文关键词。
- [ ] Hero 图正常加载。
- [ ] 完整大板没有被裁切。
- [ ] 应用图主体没有被裁掉。
- [ ] 工厂图能体现能力。
- [ ] 每张重要图片有 alt。
- [ ] 图片点击放大或图集切换正常。

## 首页

- [ ] 第一屏看得出品牌和主打材料。
- [ ] 有产品线入口。
- [ ] 有 current slabs 入口。
- [ ] 有应用场景入口。
- [ ] 有工厂能力入口。
- [ ] 有 WhatsApp / Contact CTA。
- [ ] 移动端标题不溢出。

## 产品页

- [ ] 产品信息清楚。
- [ ] 规格表清楚。
- [ ] 图集可用。
- [ ] CTA 明确。
- [ ] 相关板面或文章存在。

## 板面页

- [ ] 完整板面显示完整。
- [ ] 缩略图可切换。
- [ ] 不写死过期库存。
- [ ] 有 current lot 提示。
- [ ] 有 WhatsApp / inquiry CTA。

## 工厂页

- [ ] 有机器和加工能力说明。
- [ ] 有质检、包装、装柜说明。
- [ ] 不把图片数量当卖点。
- [ ] 页面不杂乱。

## SEO 文章

- [ ] 每篇有独立 slug。
- [ ] 有 meta description。
- [ ] 有 hero 图。
- [ ] 正文图正常加载。
- [ ] 表格不重叠。
- [ ] 有内链和 CTA。
- [ ] 有 Article JSON-LD。

## 联系与转化

- [ ] WhatsApp 正确。
- [ ] Phone 正确。
- [ ] Email 正确。
- [ ] Form endpoint 正确。
- [ ] Floating WhatsApp / Call / Top 正常。
- [ ] Contact 页面清楚列出询盘所需信息。

## 构建部署

- [ ] `pnpm test` 通过。
- [ ] `pnpm build` 通过。
- [ ] GitHub main 已推送。
- [ ] Cloudflare Pages 部署成功。
- [ ] 自定义域名 HTTPS 正常。
- [ ] Google Search Console 已添加域名。
- [ ] 已提交 `https://domain.com/sitemap.xml`。
