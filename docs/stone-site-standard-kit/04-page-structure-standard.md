# 04 Page Structure Standard

## Home

首页第一屏必须让客户马上知道：

- 这是什么品牌。
- 主打什么石材。
- 服务什么买家。
- 下一步如何询盘。

推荐模块：

1. High-end visual hero。
2. 产品线入口。
3. Current slabs / lots。
4. 应用场景。
5. 工厂能力。
6. SEO guide / articles。
7. Contact CTA。

避免：

- 纯导航页。
- 空洞口号。
- 图片很多但不知道卖什么。
- 第一屏看不出石材主体。

## Product Page

必须包含：

- 产品名称。
- 材质。
- 产地或生产方式。
- 厚度。
- 表面处理。
- 应用。
- 产品图集。
- 规格表。
- 询盘 CTA。
- 相关板面或相关文章。

图片区要支持点击切换主图或放大图。

## Slab Page

目标是促成客户询问当前 lot，而不是承诺永久库存。

必须包含：

- 完整大板图。
- 缩略图图集。
- 材质、表面、厚度、尺寸说明。
- 库存更新提示。
- WhatsApp / Request CTA。

完整大板图不裁切。

## Applications Page

用于让客户快速想象项目效果。

建议分类：

- Kitchen island。
- Countertop。
- Full-height backsplash。
- Bathroom vanity。
- Feature wall。
- Flooring。
- Commercial reception。

## Factory Page

工厂页必须建立信任，不要只堆图。

推荐顺序：

1. Quarry / block source。
2. Slab selection。
3. Gang saw / multi-wire saw。
4. Polishing / honed finish。
5. CNC / cut-to-size。
6. Dry layout / bookmatch。
7. Quality inspection。
8. Packing。
9. Container loading。
10. Export support。

## Contact Page

必须简洁、清楚、能提交询盘。

必须包含：

- WhatsApp。
- Phone。
- Email。
- Inquiry form。
- Inquiry checklist。
- Address。

询盘字段建议：

```txt
Name
Email
WhatsApp / Phone
Material
Thickness
Finish
Quantity
Destination port
Message / drawings / preferred slab photos
```

浮动按钮：

```txt
WhatsApp
Call
Top
```
