# 石材网站建站规则包

这个文件夹用于沉淀以后制作石材外贸网站的统一规则。新建花岗岩、大理石、石英岩、石英石、板材库存、工厂型供应商网站时，优先按这里执行。

## 文件说明

- `stone-site-build-rules.md`：完整建站规则，适合开新项目之前阅读。
- `stone-site-checklist.md`：执行检查清单，适合上线前逐项核对。

## 适用网站

- 石材品牌官网
- 天然石材产品目录站
- 石英石 / 人造石产品目录站
- 工厂型供应商展示站
- SEO 博客 / 指南内容站
- Cloudflare Pages / GitHub 静态部署站

## 核心原则

1. 纯静态，方便部署和维护。
2. 内容数据化，不把产品和文章硬写死在页面里。
3. 图片先规范，再建页面。
4. 产品页要服务询盘，不做漂亮但无转化的页面。
5. 每个页面都要有明确 SEO 目标和买家行动路径。
