# Site Map

## Core Pages

- `/` - Home page and collection entrance.
- `/about/` - TM Stone supplier story and capability.
- `/applications/` - Application/effect image gallery for kitchens, vanities, walls, floors, and commercial interiors.
- `/contact/` - Inquiry instructions, email, WhatsApp, and address.
- `/factory/` - Factory, quarry, warehouse, processing, dry layout, packing, and container loading gallery.
